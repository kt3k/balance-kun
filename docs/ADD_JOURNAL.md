# JOURNAL 追加カード

Journal を追加するためのカード (フォーム)

## 表示要素

- No. (固定)
- 日付
- OK ボタン
- 説明 <input type="text">
- DEBIT ラベル
- DEBIT 追加ボタン
- DEBIT タイトル入力
- DEBIT 金額入力
- DEBIT 削除ボタン (DEBIT が2個以上の場合のみ)
- CREDIT ラベル
- CREDIT 追加ボタン
- CREDIT タイトル入力
- CREDIT 金額入力
- CREDIT 削除ボタン (CREDIT が2個以上の場合のみ)

## 行動要素
- 日付を変更する
- 説明を変更する
- DEBIT を入力する
- DEBIT を追加する
- DEBIT を削除する
- CREDIT を入力する
- CREDIT を追加する
- CREDIT を削除する
