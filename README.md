# バランス君

バランスシート作成アプリ

- フリーランスエンジニア/デザイナ/コンサルタント向け
- 年次取引数1000未満の個人事業主がメインターゲット

# High level architecture

- WebView app
  - Use [straw][straw] as native bridging framework.
- Use [capsid][capsid] as UI framework.
